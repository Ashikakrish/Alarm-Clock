# ⏰ Alarm Clock

A simple and responsive Alarm Clock web application built using HTML, CSS, and JavaScript. It displays the current time and allows users to set an alarm.

## 🚀 Features

- Real-time digital clock
- Set and manage alarms
- Responsive user interface
- Easy-to-use controls

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Alarm-Clock/
├── index.html
├── style.css
└── script.js
```

## ▶️ How to Run

1. Clone or download this repository.
2. Open `index.html` in your web browser.

## 📌 Future Enhancements

- Snooze functionality
- Dark mode
- Custom alarm sounds
- 12-hour / 24-hour format toggle
